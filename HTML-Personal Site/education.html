Courses I learned at SAIT

First Semester
  • Information Technology
    • IT literacy in Linux and Windows CLI, basic database. with academic courses such as job profiles, and professional skills, and opportunities
  • Programming Essentials
    • The concept of computer programming languages as tools including problem solving techniques, overview software design methodology etc.
  • Computer Hardware and Operating System
    • Both H/W and S/W essentials of computer system
  • Introduction to Networking
    • Introduction to Networking by introducing terminology, devices, standards, OSI model. Skills learned in terms of hubs, bridges, swiches, routers and basic cabling, IP addressing and network standards.
  • Professional Communication
    • Effective ways of technical documents, reports, and presentations produced in relevant industry setting.
Second Semester
  • Object Oriented Programming Essentials
    • Object-oriented concepts, oop design concepts, GUI, polymorphism, inheritance, exception handling, interface, JDBC
  • Website Development Fundament
    • Web technology and the skills, install and configure a Web Server. Create and implement Website Applications.
  • Database Design & Programming
    • Create database application designs and implement using multiple database vendors. Learned with SQL to create, manipulate data, and retrieve information from the database.
  • Interface Design
    • The concepts related to designing user interfaces for software applications. Appropriate documentation, usability engineering, HCI design standards, screen mock-ups, creation of prototypes.
  • Business Dynamics
    • Principles of business operations in the Information Technology field with emphasis on entrepreneurship and data-analytics. Organizational effectiveness strategies in a project team with various scenarios.
Third Semester
  • Advanced Object-Oriented Programming
    • More advanced object-oriented concepts included oo design concepts, incorporating databases into software, ADT, recursion, and the concept of data structures and algorithms
  • Object Oriented Systems Analysis and Design
    • The methods of object-oriented software engineering with planning and feasibility, requirements analysis, specification, system life-cycle, prototyping, object-oriented design, documentation, and CASE tools.
  • Web Application Programming
    • Web technologies, web application development (using Java EE APIs), and web application design and architecture
  • Database Programming & Testing
    • Focus on database specific structured languages and embedding SQL into these languages. Basic programming structures, storing, triggers, writing efficient code, advanced SQL concepts, and code testing.
  • Project Preparation
    • Basic project management practices for initiating and defining the scope of a project, planning, executing project with an effective project team and successfully completing the project to the client's satisfaction
Fourth Semester
  • Capstone Project
    • Demonstrate an integration of technical skill and knowledge, professional competencies and development/execution strategies
  • Operating Systems for Software Developers
    • Using both windows and Linux, the fundamental principles and management of operating systems, mechanisms for process, memory, storage, and protection management
  • Introduction to Database Administration
    • Overview of database administration concepts and the role of the DBA using RDBMS. Database file and memory structures and management, network connectivity, database instance creation, backup and recovery.
  • Secutity for Software Developers
    • Exam program structure, web application, database security and common soft ware vulnerabilities. The creation and prevention of buffer overflows, SQL injection, the stucture of malware, flaws in encryption.
  • Emerging Trends in Technology
    • Research and investigate potential technology trends identified by industry includes identification of emerging trends, impact on different organizations, and the creation of business proposals, and utilization of peer feedback
